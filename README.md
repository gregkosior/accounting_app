# Accounting App

Local-first accounting application for analyzing CSV and XLSX financial data.
Designed to speed up accountantsâ€™ daily work by automating calculations,
summaries and visualizations.

## Author
Grzegorz Kosior

## Version
v1.0.0

## Technology stack
- Python 3.11+
- Flask
- Pandas
- Matplotlib / Plotly
- HTML / CSS
- PyWebView (desktop-ready)
- PyInstaller (EXE build)

## Key features
- Import CSV and XLSX files
- Automatic financial analysis
- Income / expense detection
- Configurable analysis settings
- Data visualization (charts)
- Export results to:
  - Excel
  - PDF
- Analysis history
- Local-first (no data sent to the internet)
- Multi-language ready (settings-based)
